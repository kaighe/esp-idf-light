# ESP-IDF Light

A set of lightweight vscode tools for ESP-IDF development.

## Features

Running the command `ESP-IDF Light: Create New Terminal` will allow you to easily start IDF Powershells for any version you have installed.

## Extension Settings

This extension contributes the following settings:

* `esp-idf-light.basePaths`: A list of paths to search of IDF versions.
* `esp-idf-light.additionalInstallations`: A list of installations for versions not found in any of the base paths.
